# Sanctum Core Landing Page

This is the official landing page for the Sanctum Core luxury underground vault project.

## Tech Stack
- Next.js with App Router
- Tailwind CSS
- Hosted on Vercel

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

Then upload concept art images to `/public`:
